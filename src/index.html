Loading Local GeoJSON Files with a Custom Base Map Demo <view>the code</view>

An example of loading local geoJSON files as layers on top of a custom base map.

This demo shows the distribution of jobs and population across Vermont.

The hexagaon data set was downloaded from the VCGI Open Data Portal.

QGIS was then used to load the shape files (.shp) and convert them to geoJSON.

To be useful, geojson needs to have its coordinates set to latitude and longitude so the files were saved using the Coordinate Reference System (CSR) EPSG:4326. File extensions were then changed to ".json" so they could be easily served

Example using:MapBox

An app hosted on Mapbox created with Mapbox-GL <view>the code</view>

I wanted a map showing the age of Burlington buildings (inspired by Age of LA).

Data Sources: A shapefile containing Vermont building outlines was found here, and property information containing the year built was found on the Burlington City Data Portal. These were further processed and joined together using Excel and qGIS.

Map Style: In mapbox I created a style and uploaded this shapefile as a tileset. Then I added the tileset as a layer, and to the right of the layer -> Style -> Color select there is a steps icon which allows you to color buildings differently according to a property.

Additional Info & Interactivity: Back in the web page where I display this map, I used JavaScript to get the map style information and use it to create a legend and show building year information.

Example using:Leaflet

One layer of data in Leaflet.js <view>the code</view>

Data Sources: A shapefile containing Vermont building outlines was found here, and property information containing the year built was found on the Burlington City Data Portal. These were further processed and joined together using Excel and qGIS.

Map Style: This was almost entirely styled with leaflet.js built-in options, and one custom function getColor, which applies a color to the building based on its YearBuilt property.

Additional Info & Interactivity: Leaflet.js has built in functions for creating controls, which I used. The onEachFeature function adds the highlighting and year display for each building.

Multiple layers in Leaflet.js <view>the code</view>

Data Sources: This uses the same data as the other Burlington Buildings apps. I create seven layers by using a filter function to extract the different segments from the same geojson data.

Map Style: Each layer is given a different color as it is added to the map. Most of the styling is the leaflet.js defaults.

Additional Info & Interactivity: The layer toggling legend/control was created entirely by leaflet.js.
